JavaScript Objects
==================

#### Literal Notation

``` javascript

var myObj = { "name" : "myname",
              "age"  : 30
        
};

``` 


accessing objects

    console.log(myObj.name);
    $("#main").append(myObj.city);   # using jquery selector, dot notation
    $("#main").append(myObj(city["Somewheresville"]);  # bracket notation

#### Dot Notation

``` javascript
    
var myObj = {};
myObj.city = "Somewhere City";

```


#### Bracket Notation

``` javascript

var myObj = {};
myObj["name"] = "Name";

```
