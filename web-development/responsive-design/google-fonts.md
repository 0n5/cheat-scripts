Google Fonts
============


Navigate to: [Google Fonts](https://www.google.com/fonts)<br>
Find a font to install

#### Linking

In HTML

``` html

<head>
	<link href='https://fonts.googleapis.com/css?family=Poiret+One' rel='stylesheet' type='text/css'>
</head>

```

In CSS

``` css

font-family: 'Poiret One', cursive;

```
