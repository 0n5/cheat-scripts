Node.js + Yeoman cheats
====================================


<h4>Install:</h4>
<pre>
# requires NVM installed properly and not as root
$ npm install -g yo
$ mkdir site
$ cd site
$ npm install -g generator-webapp
$ yo webapp
</pre>

<h4>Add ip address (if applicable) to hosts file:</h4>
<pre>
$ nano /etc/hosts
</pre>

Access website at localhost:9000
