#### ngAnimate


##### Installation 

in index.html

<script src="//ajax.googleapis.com/ajax/libs/angularjs/X.Y.Z/angular-animate.js"></script>


in app.js

var [MODULE] = angular.module('MODULE', ['ngAnimate']);


##### Usage

create css/animations.css file and add link to index.html
