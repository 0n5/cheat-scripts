React Installation
==================

#### Prerequisites

Install Yarn, Install Nodejs via NVM (optionally use Yarn instead of NPM)

    
    nvm install v8.1.2
    
#### create-react-app

Auto installs React, Babel, ESLint, Webpack and scaffolds

    npm install -g create-react-app
    create-react-app REACT_PROJECT

Run server

    cd REACT_PROJECT
    npm start
    
Run tests

    npm test
